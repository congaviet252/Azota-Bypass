TÍNH NĂNG

Tính năng chính

Script này cung cấp nhiều tính năng mạnh mẽ để hỗ trợ bạn trên nền tảng Azota.vn

Tự động trả lời câu hỏi

Tự động phân tích và trả lời các câu hỏi trắc nghiệm và tự luận sử dụng trí tuệ nhân tạo tiên tiến.

Gợi ý đáp án thông minh

Hiển thị gợi ý đáp án cho câu hỏi đang hiển thị, kèm theo giải thích ngắn gọn dễ hiểu cho mọi câu hỏi.

Bảng điều khiển thông minh

Bảng điều khiển script có thể được kéo và thả tự do trên màn hình, giao diện thân thiện với người dùng.

Bộ nhớ ngữ cảnh nâng cao

Lưu lại lịch sử câu hỏi và câu trả lời để tạo ngữ cảnh cho AI, nâng cao chất lượng câu trả lời tự luận.

Hệ thống ghi log chi tiết

Ghi lại mọi hoạt động của script, bao gồm câu hỏi, câu trả lời và các sự kiện khác để dễ dàng theo dõi.

Tùy chỉnh API Key

Cho phép người dùng nhập và thay đổi Voiceim V1 API Key để sử dụng công nghệ AI mạnh mẽ nhất hiện nay.

HƯỚNG DẪN

Cách sử dụng

Chỉ 4 bước đơn giản để bắt đầu sử dụng công cụ mạnh mẽ này!

1

Cài đặt Tampermonkey

Cài đặt tiện ích mở rộng Tampermonkey cho trình duyệt của bạn (Chrome, Firefox, Edge, v.v.).

Tải Tampermonkey
2

Cài đặt script

Sao chép mã script và dán vào trình soạn thảo script của Tampermonkey để kích hoạt tính năng.

Tampermonkey Dashboard > + New script
3

Nhập Voiceim V1 API Key

Nhấn nút "API Key" trên bảng điều khiển và nhập API Key của bạn để kết nối với AI.

Tạo Voiceim V1 API Key
4

Sử dụng tính năng

Sau khi cài đặt xong, bạn có thể sử dụng các tính năng mạnh mẽ sau:

  • Nhấn "Bắt đầu Auto (Huỳnh Dương)" để tự động trả lời tất cả câu hỏi.
  • Nhấn "Huỳnh Dương gợi ý" để xem gợi ý đáp án cho câu hỏi hiện tại.
  • Nhấn "Hiện Log" để xem log chi tiết về quá trình xử lý.

YÊU CẦU

Yêu cầu hệ thống

Để sử dụng script một cách hiệu quả, hãy đảm bảo đáp ứng các yêu cầu sau

  • Trình duyệt có cài đặt Tampermonkey

    Hỗ trợ các trình duyệt phổ biến như Chrome, Firefox, Edge và Safari - đảm bảo trải nghiệm tốt nhất.

  • Voiceim V1 API Key hợp lệ

    Bạn cần có một Voiceim V1 API Key để kết nối với trí tuệ nhân tạo tiên tiến nhất hiện nay.

  • Kết nối Internet ổn định

    Cần có kết nối Internet ổn định để script có thể giao tiếp với Voiceim V1 API và xử lý dữ liệu.

  • Tài khoản Azota.vn

    Bạn cần có tài khoản Azota.vn để truy cập nền tảng và sử dụng các tính năng của script.

LƯU Ý

Lưu ý khi sử dụng

Để có trải nghiệm tốt nhất, vui lòng lưu ý các điểm sau khi sử dụng script

  • API Key của bạn được lưu trữ cục bộ và chỉ được sử dụng để giao tiếp với Voiceim V1 API.

  • Tính năng tự động trả lời dựa trên công nghệ AI nên độ chính xác có thể không đạt 100%, đặc biệt với các câu hỏi phức tạp.

  • Script được phát triển với mục đích hỗ trợ học tập, giúp người dùng hiểu bài và tiết kiệm thời gian.

  • Script hỗ trợ các câu hỏi phổ biến trên Azota, các câu hỏi có định dạng đặc biệt hoặc phức tạp có thể cần điều chỉnh thủ công.

  • Tính năng "Huỳnh Dương gợi ý" là lựa chọn an toàn nhất khi bạn muốn tham khảo đáp án và lời giải chi tiết.
